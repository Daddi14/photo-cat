# Pubblicazione

Questa pagina è per chi mantiene il progetto.

## Checklist release

1. Aggiorna `VERSION`.
2. Aggiorna `pyproject.toml`.
3. Esegui i controlli di sintassi.
4. Verifica i launcher su Windows e macOS/Linux.
5. Prepara l’archivio della release.
6. Pubblica le note di rilascio.

Gli utenti finali devono continuare a usare `START_WINDOWS.bat` o `START_UNIX.sh`.
