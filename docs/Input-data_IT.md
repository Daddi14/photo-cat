# Dati di input

PHOTO-CAT usa file CSV.

## Catalogo

Il catalogo deve contenere, o deve poter mappare tramite GUI, queste colonne:

- `source_id`
- `ra`
- `dec`
- `phot_g_mean_mag`

I nomi delle colonne sono case-sensitive.

## Target

I target possono essere forniti tramite CSV oppure come lista manuale di source ID nella configurazione.
