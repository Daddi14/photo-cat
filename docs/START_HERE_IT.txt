PHOTO-CAT - Avvio rapido

Windows
1. Estrarre lo ZIP della release.
2. Aprire la cartella photo-cat estratta.
3. Fare doppio click su START_WINDOWS.bat.

macOS / Linux
1. Estrarre lo ZIP della release.
2. Aprire il Terminale nella cartella photo-cat estratta.
3. Eseguire:
   sh START_UNIX.sh

Dopo l'avvio
1. Selezionare il Catalog CSV.
2. Verificare che i nomi delle colonne corrispondano esattamente all'header del CSV.
3. Controllare i percorsi generati per target e output.
4. Cliccare Save + run.

Note
- I nomi delle colonne sono case-sensitive.
- Il primo avvio può richiedere alcuni minuti per l'installazione delle dipendenze.
- Se la cartella del progetto viene spostata, PHOTO-CAT può ricostruire automaticamente l'ambiente locale .venv.
