# PHOTO-CAT Wiki

PHOTO-CAT is a local tool for photometric contamination analysis.

It builds a neighbour index from an astronomical catalogue, queries selected targets, and writes contamination results to JSON.

## What PHOTO-CAT includes

- Windows, macOS, and Linux launchers.
- Graphical configuration interface.
- Local virtual environment management.
- Project-local runtime fallback.
- CSV catalogue and target handling.
- Neighbour-index build stage.
- Contamination query stage.
- JSON output.
