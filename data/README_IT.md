# Dati di esempio

Questa cartella contiene piccoli CSV di esempio per provare PHOTO-CAT subito dopo il download.

File inclusi:

- `example_catalog.csv`
- `example_targets.csv`

Questi file sono solo esempi. Sostituiscili con cataloghi e target reali nella configurazione grafica.

Le colonne di esempio predefinite sono:

- `source_id`
- `ra`
- `dec`
- `phot_g_mean_mag`

I file di output generati dalla pipeline non devono essere committati.
